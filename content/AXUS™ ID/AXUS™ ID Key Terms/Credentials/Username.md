[[Username]] is a unique string identifier for every user. Unlike [[AUID]], [[Username]] can be changed at any time by user.

[[Username]] should be used only in user interface, not inside systems as the way to identify user. To identify a user [[AUID]] must be used because it is unchangeable. But UI should not even contain [[AUID]], because an ordinary user should not know even that [[AUID]] exists.
