## Evaluation
Maximum number of points is `100`.
- `5` points for **test 1** (`60` minutes).
- `9` points for **test 2** (on `11th` week, `4` problems to solve).
- `-2` to `4` points for activity on **seminar**s (`-0.5` to `1` points for a **seminar**). Each problem solution on **seminar** must be named in format `<StudentID>_<Year>_<SeminarNumber>_<ProblemNumber>.c` (e.g. `1254154_2022_1_1.c`).
- `35` points for your **project** (`32.5` for the **project**, `2.5` for **project defense**) (see [[#Project]]).
- `47` points for the final **exam**.

If you got $\le 28$ points for `2` **test**s, **seminar**s and **project** and cumulative, you can’t write the **exam**. If you got $\le 15\%$ points for the final **exam**, you can’t complete the course successfully.

Points-to-Marks table is the same as [[Default FIIT STU Points-to-Marks table]].
